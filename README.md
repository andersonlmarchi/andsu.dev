# andsu.dev - Portfolio

Portfolio pessoal com estilo de terminal Linux, inspirado no design minimalista e moderno.

## 🚀 Tecnologias

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion

## 📦 Instalação

```bash
npm install
```

## 🏃 Executar

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## 📝 Personalização

### Adicionar Projetos

Edite o arquivo `components/Projects.tsx` e adicione seus projetos no array `projects`.

### Adicionar Imagens

Coloque as imagens dos projetos em `public/projects/` e atualize o caminho no array de projetos.

### Adicionar Códigos

Edite o arquivo `components/CodeShowcase.tsx` para adicionar mais exemplos de código.

## 🎨 Estilo

O portfólio usa um tema de terminal Linux com:
- Fundo preto (#000000)
- Texto verde (#00ff41)
- Fonte monospace
- Efeitos de glow e animações
